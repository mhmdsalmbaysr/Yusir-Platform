# منصة يُسر — دليل المشروع

**يُسر** هي منصة تفاعلية تعرض المتاجر والمنتجات على خريطة اليمن. تعمل بالكامل على المتصفح (Static SPA) ولا تحتاج إلى خادم خلفي.

---

## هيكلة الملفات

```
map3/
├── index.html              # الصفحة الرئيسية (الخريطة + المتصفح)
├── app.js                  # منطق الخريطة الرئيسية وعرض المتاجر
│
├── admin.html              # لوحة الإدارة (إضافة المتاجر وبيانات الحقل)
├── admin.js                # منطق إدارة المتاجر والمواقع
│
├── login.html              # صفحة الدخول الموحّدة (تاجر / سوبر أدمن)
├── login.js                # منطق تسجيل الدخول والجلسات
│
├── merchant.html           # لوحة التاجر (إدارة المتجر والمنتجات)
├── merchant.js             # منطق لوحة التاجر
│
├── super-admin.html        # لوحة السوبر أدمن (إدارة التجار)
├── super-admin.js          # منطق السوبر أدمن
│
├── config.js               # بيانات الدخول الافتراضية
│
├── data/
│   ├── yem_admin1.geojson  # حدود وأسماء المحافظات
│   ├── yem_admin2.geojson  # حدود وأسماء المديريات
│   ├── yem_field_data.geojson  # الأحياء والحارات
│   └── stores.geojson      # المتاجر والمنتجات
│
└── css/
    ├── style.css           # تنسيقات الصفحة الرئيسية
    ├── dashboard.css       # تنسيقات لوحتي التحكم
    └── admin.css           # تنسيقات لوحة الإدارة
```

---

## التشغيل

1. افتح المجلد في **VS Code**
2. شغّل **Live Server** (أو أي خادم HTTP محلي)
3. الصفحات المتاحة:

| الصفحة | الرابط |
|---|---|
| الخريطة العامة | `index.html` |
| لوحة الإدارة | `admin.html` |
| دخول موحّد | `login.html` |
| لوحة التاجر | `login.html?redirect=merchant` |
| لوحة السوبر أدمن | `login.html?redirect=superadmin` |

> **ملاحظة:** لا تعمل `file://` المباشرة — يجب استخدام خادم HTTPS محلي (Live Server أو ما يشبهه).

## بيانات الدخول

- **سوبر أدمن:** `superadmin` / `yusir@2024`
- **التجار:** يُنشئون من لوحة السوبر أدمن ويُخزّنون في `localStorage['yusir_merchants']`

---

## آلية العمل

### الخريطة
تعتمد على **Leaflet 1.9.4** مع طبقة OpenStreetMap:
- **المحافظات:** حدود رمادية وأسماء واضحة عند التكبير ≥ 7
- **المديريات:** حدود خفيفة وأسماء عند التكبير ≥ 9
- **الأحياء والحارات:** رقاقات 🏘️ حي / 📌 حارة عند التكبير ≥ 11
- **المتاجر:** أيقونات حمراء/رمادية مع نافذة منبثقة تعرض المنتجات
- مقيدة بإحداثيات اليمن فقط (`maxBounds` + قناع يخفي الدول الأخرى)

### الجلسات
- **sessionStorage** لجلسات الدخول (تُمسح بإغلاق التبويب)
- تحويلة تلقائية إلى `login.html` عند فقدان الجلسة

### التخزين
| البيانات | الموقع |
|---|---|
| حسابات التجار | `localStorage['yusir_merchants']` |
| بيانات المتجر والمنتجات (للتاجر) | `localStorage['yusir_store_<id>']` |
| المتاجر الرسمية | `data/stores.geojson` |
| الأحياء والحارات | `data/yem_field_data.geojson` |
| الجلسات النشطة | `sessionStorage` |

## سير العمل

### إضافة متجر (in `admin.html`)
1. انقر على الخريطة لتحديد الموقع
2. أدخل بيانات المتجر
3. أضف المنتجات
4. احفظ ← يصدر `stores.geojson`

### إدارة التاجر (in `merchant.html`)
- تعديل بيانات متجره (اسم، شعار، ...)
- إضافة/حذف/تعديل المنتجات مع رفع صور من الجهاز
- عرض الطلبات (إن وجدت)

### إدارة السوبر أدمن (in `super-admin.html`)
- إضافة/حذف التجار
- مراقبة المتاجر

### بيانات الحقل (in `admin.html`)
1. اختر المديرية من القائمة
2. اختر النوع (حي/حارة)
3. انقر على الخريطة لتحديد الموقع
4. احفظ ← يصدر `yem_field_data.geojson`

## تخصيص إضافي

- **ألوان الخريطة:** غيّر `.gov-label` و `.dist-label` و `.field-chip` في ملفات CSS
- **الحدود:** عدّل `maxBounds` في `app.js` و `admin.js`
- **صورة افتراضية للمنتجات:** غيّر الرابط في `merchant.js:addProduct()`
- **بيانات السوبر أدمن:** عدّل `config.js`

---

> المشروع مبني لبيئة **Live Server** ولا يدعم Backend. جميع العمليات طرفية (Client-side).
